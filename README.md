
# Rent-car-back

Proyecto rent-car utilizando express, soap y prisma 


## Installation

Install rent-car-back with npm

```bash
  cd rent-car-back
  npm i
```
Set-up prisma and db

```bash
  npx prisma migrate status
  npx prisma migrate dev
  npx prisma generate
```

Usage
```bash
  npm run dev
```
